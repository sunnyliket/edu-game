## 🏜️ STAGE 2 상세 설계: 좀비와 모래폭풍의 사막 (물체의 운동)

### 기본 설정
- **주제 과목**: 물체의 운동 1~9번 + 산과 염기 6~11번 = **총 15문제**
- **시점**: 3인칭 3D 캐릭터 (마을 주민 모델)
- **환경**: 끝없는 모래사막
- **선행 조건**: 스테이지 1 클리어 (과학 초능력 1 보유 상태)
- **배경 음악**: 이집트/피라미드 챌린지 테마 음악 (반복 재생)

### 🎯 NPC 상호작용
- **주민 대사**: "(스테이지 2를 클리어 하러 가자)"
- **진입 트리거**: 버튼 클릭 → 화면에 *스테이지 2: 과학의 마을* 타이틀 팝업 표시

### ❤️ 체력 시스템 UI
- 화면 하단에 **체력: ❤❤❤❤❤ (5개)** UI 상시 표시
- 피해를 받으면 체력 감소, 0이 되면 게임 오버

### 🌪️ 특수 환경 기믹
1. **모래폭풍** (1분마다 발생)
   - 피격 시 3초 동안 효과 적용:
     - 이동 속도 저하
     - 채굴 속도 저하
     - 시야 극도로 어두워짐 (화면 암전 효과)

2. **밤 좀비 스폰**
   - 밤이 되면 좀비가 스폰됨
   - 좀비 공격력: 1 (체력 1 감소)
   - 플레이어에게 접근하면 공격

### 🏆 스테이지 클리어 조건
- 15문제 전부 정답 시:
  - 화면 가득 **"복구 완료"** 텍스트 표시
  - 주민이 인벤토리에 **에메랄드 2개** 지급
  - 다음 구역(피라미드) 개방

## 🔺 피라미드 챌린지 (스테이지 2 보너스)

### 진입 방식
- 화면에 **3개의 피라미드** 중 1개 선택 가능
- 선택 후 피라미드 내부로 이동
- 내부 진입 시 **1인칭 시점**으로 자동 전환

### 🌀 미로 탐험
- 피라미드 내부는 **1인칭 미로** 구조
- 진입 즉시 화면에 **"유리병을 찾으세요"** 텍스트 표시
- 미로 중간중간에 등장하는 위험 요소:
  - **전갈** (접촉 시 피해)
  - **모래폭풍** (시야 차단)

### 🍾 유리병 이벤트
- 미로 내 유리병 발견 → **E 키** 눌러 내부 쪽지 확인
- 쪽지 확인 즉시 이벤트 발동:
  - **피라미드 붕괴 연출** 시작
  - **거대 전갈 보스** 등장 및 추격 시작

### 🏃 탈출 시퀀스 (보스 추격)
- 플레이어는 전갈에게 쫓기며 출구를 향해 탈출
- **낙석 경고 시스템**:
  - 바닥에 **빨간 원** 표시
  - 화면 상단에 **"돌이 떨어져요"** 경고 텍스트
  - 잠시 후 해당 위치에 **돌 낙하** (밟으면 피해)
- 거대 전갈에 공격당하면 **HP 10 감소** (HP는 최대 200 기준)
