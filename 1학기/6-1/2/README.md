# 과학 방탈출 숲속 섬 게임

6학년 1학기 과학 문제를 풀며 포인트를 모으고, 포인트로 캐릭터 장비, 집, 동물, 씨앗, 탈것을 구매해 나만의 숲속 섬을 꾸미는 웹 게임입니다.

## 실행 방법

1. `index.html`을 웹 브라우저에서 엽니다.
2. 이름과 중복되지 않는 아이디를 입력합니다.
3. 과학 방탈출 미션을 풀어 포인트를 얻습니다.
4. 섬으로 이동해 마우스 클릭 또는 키보드 방향키/WASD로 캐릭터를 움직입니다.

## 주요 보완 기능

- 고유 아이디 생성과 로컬 중복 검사
- 9x9로 확장된 둥근 숲속 섬
- 숲마을풍 오리지널 둥근 캐릭터 스타일
- 3D 섬 시각화와 바다 배경
- 친구 추가, 친구 집 방문, 옷 거래 제안
- 장비 등급별 가격과 이동/포인트 보너스
- 집 업그레이드, 섬/집 꾸미기, 동물 구매
- 물뿌리개가 필요한 씨앗 심기와 30초 성장
- 비싼 오토바이와 자동차 구매 및 탑승
- 25% 확률 넌센스 퀴즈
- 1~4단원 과학 미션 총 56개

## 파일 구조

```text
index.html      # 화면 구조
style.css       # UI와 캐릭터/섬 스타일
character.js    # 캐릭터, 상점, 친구, 거래, 농사 저장 데이터
missions.js     # 미션 목록
quiz.js         # 과학/넌센스 퀴즈 데이터
visual3d.js     # 3D 섬 시각화
game.js         # 게임 진행 로직
```
