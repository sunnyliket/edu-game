# 가게 차리기 오키

돈을 벌기 위해 디저트 가게를 연 플레이어가 가게를 청소하고, 마트에서 문제를 풀어 재료를 산 뒤, 첫 손님에게 초코칩 쿠키 5개를 만들어 주는 클릭형 웹 게임입니다.

## 실행

`index.html`을 브라우저에서 열면 바로 시작할 수 있습니다.

## 진행 흐름

1. 인트로 보기
2. 지저분한 가게 청소하기
3. 마트에서 버터, 박력분, 초코칩, 계란 문제 풀기
4. 문제를 모두 맞히면 자동으로 가게로 돌아오기
5. 첫 손님 주문: 초코칩 쿠키 5개
6. 레시피 순서대로 쿠키 만들기
7. 손님에게 전달하고 첫 장사 성공
