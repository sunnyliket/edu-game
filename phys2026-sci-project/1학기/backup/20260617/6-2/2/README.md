# 가게 차리기

과학 퀴즈가 들어간 3인칭 디저트 가게 게임입니다.

## 실행

`index.html` 파일을 브라우저로 열면 바로 시작할 수 있습니다.

## 조작

- WASD 또는 방향키: 플레이어 이동
- E: 가까운 재료, 손님, 조리도구와 상호작용
- 마우스 클릭: 가까운 물건 클릭 상호작용

## 진행

1. 마트에서 버터, 강력분, 초코칩, 계란을 찾습니다.
2. 재료를 얻으려면 과학 퀴즈를 맞혀야 합니다.
3. 재료를 모두 모으면 자동으로 가게로 돌아갑니다.
4. 첫 손님의 초코칩 쿠키 5개를 만들며 과학 퀴즈를 풉니다.
5. 단체 손님 주문이 오면 추가 과학 퀴즈와 쿠키 제작이 이어집니다.
